# Week 1 - JavaScript Basics

This folder contains introductory JavaScript code and exercises. The scripts focus on fundamental programming concepts such as variables, operators, conditional statements, arrays, objects, and functions.

## Contents

Here is a brief overview of the scripts included in this folder:

- **`arrayOfObjects.js`**: Demonstrates the creation and manipulation of arrays containing JavaScript objects.
- **`function.js`**: Examples of defining and calling JavaScript functions.
- **`greatestOf3.js`**: Logic to find the greatest among three given numbers.
- **`largestNum.js`**: Finds the largest number, likely from an array or a set of inputs.
- **`smallestNum.js`**: Logic to determine the smallest number.
- **`sumArray.js`**: Calculates the sum of all elements within an array.
- **`sumOfMarks.js`**: Calculates the total sum of marks, typically used for grading logic.
- **`ternary.js`**: Examples using the ternary operator (`condition ? exprIfTrue : exprIfFalse`).
- **`ternaryOP.js`**: Further usage or variations of the ternary operator.
- **`typeOf.js`**: Demonstrates the use of the `typeof` operator to determine the data type of variables.

## Usage

You can run any of these files using Node.js in your terminal. For example:

```bash
node sumArray.js
```
