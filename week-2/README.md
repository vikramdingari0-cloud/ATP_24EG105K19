# Week 2: Levelling Up with JavaScript

Welcome to Week 2! After getting with the basics in Week 1, this week we get into the more parts of JavaScript. We explored how to handle data like a pro, work with asynchronous tasks, and write cleaner,  modern code.

##  What's Inside?

Here’s a breakdown of what we covered this week:

- **`advOpsArray.js`**: This is where the real power lies! We explored "Advanced Array Operations" like `map`, `filter`, `reduce`, `find`, and `sort`. It's all about transforming and querying data without messy loops.
- **`advOpsObjects.js`**: Taking objects to the next level. We looked at more complex ways to interact with and manage our data structures.
- **`collections.js`**: A refresher and deep dive into how we group data. We covered Arrays, Objects, and the very common "Array of Objects" pattern that you'll see everywhere in web development.
- **`dataTypesExs.js`**: Some hands-on exercises to make sure we really understand the different types of data JS handles.
- **`destructuring.js`**: A super handy ES6 feature! It shows how to "unpack" values from arrays and properties from objects into distinct variables in a single line.
- **`errorHandle.js`**: (Work in progress) This is where we learn how to make our code "bulletproof" by handling errors gracefully so the whole app doesn't crash.
- **`inheritance.js`**: We stepped into the world of Classes. We learned how one class can "inherit" features from another, making our code more reusable and organized.
- **`opsOnArray.js`** & **`opsOnObjects.js`**: Essential practice files for the bread-and-butter operations we do every day with arrays and objects.
- **`parameter.js`**: Ever wondered how functions can take any number of arguments? We explored **Rest Parameters** (`...args`) here!
- **`promise.js`** & **`promises.js`**: The heart of modern JS. We learned how to handle things that take time (like fetching data from a server) using Promises, `.then()`, and the sleek `async/await` syntax.
- **`spreadOp.js`**: Using the "magic dots" (`...`) to spread elements of an array or object into new ones. It’s like copy-pasting, but cooler.

##  Subdirectories

- **`/ADV_JS`**: This folder contains more advanced concepts like Synchronous vs. Asynchronous execution, Timer functions (`setTimeout`, `setInterval`), and how HTTP requests work.
- **`/Assignments`**: The real-world test! This folder is packed with practical challenges like building a Shopping Cart, an eCommerce logic, a To-Do list, and a temperature tracker. These files bring all our concepts together.

##  Conclusion for Week 2

Week 2 was a major milestone! We moved from "how do I write a loop?" to "how do I write elegant, efficient, and scalable code?". By mastering things like **Promises**, **Destructuring**, and **Higher-Order Array Methods**, we've started thinking like true JavaScript developers. The logic is getting sharper, and the code is getting cleaner.
