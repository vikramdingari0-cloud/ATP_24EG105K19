# Week 4: Introduction to HTML and CSS

This directory contains introductory exercises for building web pages using HTML and CSS. The focus this week was on understanding document structure, common tags, lists, and external styling.

## Files Overview

### 1. web1.html
A basic HTML structure demonstrating the use of:
- Headings (`<h2>`) and Paragraphs (`<p>`)
- Text formatting (`<b>`, `<i>`)
- Hyperlinks (`<a>`)
- Unordered lists (`<ul>`, `<li>`)

### 2. web2.html
A practical example of **Nested Lists**, showcasing how to organize hierarchical information (like Fruits and Vegetables) using `<ul>` inside another `<li>`.

### 3. web3.html
A complete profile page for "Pochi the Cat." This file integrates:
- External CSS linking.
- Image embedding (`<img>`).
- Ordered lists (`<ol>`).
- A container-based layout using `<div>`.

### 4. style.css
The external stylesheet for `web3.html`. It covers:
- Body background and font-family.
- Layout design using a centered `.box` container.
- Image positioning (floating right) and sizing.
- Basic typography and footer styling.

## Conclusion
This week marks the transition from pure JavaScript logic to visual web development. We've learned how to structure content meaningfully and apply styles to create clean, readable layouts.
