# Week 5: HTML Layouts & Tailwind CSS

Welcome to **Week 5** of the SUNTEK JavaScript and Web Development series! This week, the focus shifted from core JavaScript to advanced HTML structures, CSS layouts (Flexbox), and an introduction to **Tailwind CSS**.

## Project Overview

This directory contains several exercises designed to master table structures, responsive layouts, and modern utility-first CSS.

###  File Descriptions

- **[blog.html](blog.html)**: A complete blog-style landing page featuring a responsive navigation bar, search input, and dynamic article cards.
- **[layout.html](layout.html)**: A Flexbox-based layout demonstration using cards to showcase nature themes (Mountains, Rivers, Valleys) with embedded images.
- **[table1.html](table1.html)**: A foundational table exercise focused on mastering the `colspan` attribute for complex header structures.
- **[table2.html](table2.html)**: A detailed student grade report table utilizing both `rowspan` and `colspan` to organize academic data efficiently.
- **[table3.html](table3.html)**: A stylized "Users" table containing profile pictures and details for famous athletes, demonstrating how to style tables with custom CSS classes.
- **[tailwind/src/test.html](tailwind/src/test.html)**: An introductory exploration of **Tailwind CSS**, implementing utility classes for gradients, rounded borders, and responsive flex containers.

##  Key Concepts Learned

- **Advanced HTML Tables**: Mastering complex data representation using `rowspan` and `colspan`.
- **Flexbox Layouts**: Creating responsive, multi-column layouts without traditional floats.
- **Utility-First CSS**: Setting up and using Tailwind CSS to rapidly style components with predefined classes.
- **UI/UX Design**: Building clean, readable interfaces like blog feeds and data tables.
